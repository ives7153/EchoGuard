**Gateway 接线表**

| 模块                  | 引脚       | 接到ESP32-s3         | 说明                       |
| ------------------- | -------- | ------------------ | ------------------------ |
| Ra-02 / SX1278 LoRa | VCC      | 3V3                | 只能接 3.3V，禁止接 5V          |
| Ra-02 / SX1278 LoRa | GND      | GND                | 必须与 ESP32-S3 共地          |
| Ra-02 / SX1278 LoRa | NSS / CS | GPIO10             | LoRa SPI 片选              |
| Ra-02 / SX1278 LoRa | SCK      | GPIO12             | SPI 时钟                   |
| Ra-02 / SX1278 LoRa | MOSI     | GPIO11             | ESP32-S3 发送到 LoRa        |
| Ra-02 / SX1278 LoRa | MISO     | GPIO13             | LoRa 返回到 ESP32-S3        |
| Ra-02 / SX1278 LoRa | DIO0     | GPIO4              | LoRa 接收完成中断 / 状态检测       |
| Ra-02 / SX1278 LoRa | RST      | GPIO21             | LoRa 硬件复位                |
| LoRa 天线             | ANT      | Ra-02 天线座          | 上电和发射前必须接好天线             |
| USB 数据线             | USB      | ESP32-S3 开发板 USB 口 | 供电、烧录、串口输出，上位机波特率 115200 |
| 去耦电容 C1             | 10uF     | Ra-02 VCC 与 GND 之间 | 靠近 LoRa 模块放置             |
| 去耦电容 C2             | 0.1uF    | Ra-02 VCC 与 GND 之间 | 靠近 LoRa 模块放置             |
